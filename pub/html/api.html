API

Programmer Object (infobox & codebox)

Attribute
Type
Description
infobox.def (required)
String
the programming definition associated with the target keyword. e.g. "if"
infobox.explanation
String
the explanation of the programming definition.
e.g. "Tests a given condition and returns one value for a TRUE result and another value for a FALSE result"
infobox.example
Array of strings and subarrays
an example code block. The first element should be a string that explains what does the following code do.
e.g. ["For example, the following will print each element in students ('paul', 'lily', and 'jim') ", "students = ['paul', 'lily', 'jim']\n", "for student in students:\n" ," print(student)"]
infobox.syntax
Array of strings and subarrays
the syntax of the related programming definition. Keywords should be contained in a nested array seperately.
e.g. [["if"], " condition expression:\n", " result when condition is satisfied\n", ["else"], "\nresult when condition is NOT satisfied\n"]
codebox.question
Array of strings and subarrays
a question. Keywords should be contained in a nested array seperately.
e.g. ["Print ", ["Hello"], " if word ", ["apple"], " contains character 'a', else print ", ["Bye"]]
codebox.code_templet
Array of strings and subarrays containing a string represents the length of required user input
the code template for question. Fields required to be filled by users should be presented by an array, filled with the required length of user input.
e.g. [[2], "'a' in 'apple':\n", " print('Hello')\n", [4],":\n", " print('Bye')"]
codebox.answer
(need to provide codebox.code_templet as well)
Array of strings
answers for the question. Should match the required input fileds in code_templet
e.g.["if", "else"]
type
the type of this Programmer object
will automatically analyze infobox.def and assign a type to this Programmer object

Example

order: def, explanation, example, syntax, question, code_templet, answer

    const userprogrammer = new Programmer('user', 'explanation',
    ['example header', 'example line1\n', 'example line2\n', 'example line3\n'],
    [['keyword1'], ' syntax line1\n',['keyword2'], '     syntax line2\n'],
    [['keyword1'], ' question1'],
    [['4'], ' templet line1\n', '    templet line2\n', ['4'],'   templet line3\n'],
    ['ans1', 'ans2'])

Functions

Function name
Required Input
Description
makeInfoBox
target position, type
make an infobox, containing defintion, explanation, example and syntax. e.g.
const target = document.querySelector('.if-block')
const ifprogrammer = new Programmer('if')
ifprogrammer.makeInfoBox(target, ifprogrammer.type)
Important: please make the keyword be a innertext of hyperlink with classname "target", with empty string link.e.g.
<div class ='if-block'><a class='target' href="">If</a></div>
changeExplanation
newContent (String)
change the explanation of infobox.
e.g. ifprogrammer.changeExplanation("new explanation for if")
changeExample
newContent (Array of strings)
change the explanation of infobox. The first element should be the explanation of this example.
e.g. ifprogrammer.changeExample(['new example header','new example line1\n'])
changeSyntax
newContent (Array of strings and subarrays)
change the syntax of infobox.
e.g. ifprogrammer.changeSyntax([['keyword1'], ' syntax line1\n'])
makeCodeBox (better to use with makeResultBox)
target position, type
make a codebox. e.g.
const target = document.querySelector('.if-block') const ifprogrammer = new Programmer('if') ifprogrammer.makeCodeBox(target, ifprogrammer.type) Important: please make the keyword be a innertext of hyperlink with classname "target", with empty string link.e.g.
<div class ='if-block'><a class='target' href="">If</a></div>
makeResultBox (use with makeCodeBox)
type, codebox.answer
create the result area of codebox
e.g.
const target = document.querySelector('.if-block')
const ifprogrammer = new Programmer('if')
ifprogrammer.makeCodeBox(target, ifprogrammer.type)
ifprogrammer.makeResultBox
(ifprogrammer.type, ifprogrammer.codebox.answer)
changeQuestion
newContent (Array of strings and subarrays)
change the question of codebox.
e.g. ifprogrammer.changeQuestion([' question1\n', ['keyword1']])
changeTemplet
newContent (Array of strings and subarrays containing a string represents the length of required user input)
change the code templet of codebox.
e.g. ifprogrammer.changeTemplet(['templet line1\n', ['4']])
changeExercise
newContent (Dictionary contains two element. One key is "code" (code template), its value is array of strings and subarrays containing a string represents the length of required user input. Another key is "answer", its value is array of strings.)
change the answer of codebox.
e.g. ifprogrammer.changeExercise({'code':['templet line1\n', ['6']], 'answer':['newans']})
